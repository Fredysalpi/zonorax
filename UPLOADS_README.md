# Carpeta de Uploads

Esta carpeta contiene los archivos subidos por los usuarios.

## Estructura

- `profiles/` - Imágenes de perfil de usuarios

## Nota

Los archivos en esta carpeta NO se suben a Git (ver `.gitignore`).
Al clonar el repositorio, asegúrate de crear esta estructura de carpetas manualmente o el servidor la creará automáticamente al iniciar.
